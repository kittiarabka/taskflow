# TaskFlow Pure PHP

todo лист

Стек:
- PHP 8+
- MySQL
- PDO
- ООП
- MVC структура
- HTML/CSS/JS

Запуск:
1. Установить XAMPP или MAMP.
2. Скопировать папку в htdocs (XAMPP) или папку сервера.
3. Создать базу данных taskflow.
4. Импортировать database/taskflow.sql.
5. Настроить config/config.php.
6. Открыть http://localhost/TaskFlow_Pure_PHP/public/

Функционал:
- создание задач
- удаление задач
- API для задач
